## Installation
Clone the repository
```
git -clone https://github.com/Celentroft/token-checker.git
```
Install requirements
```
pip install -r requirements.txt (or use install.bat)
```
Input Token in file
```
Go into folder "data" and input token in "tokens.txt
```
Launch script
```
python main.py
```
